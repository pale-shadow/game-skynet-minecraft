## 2. Table: `skynet_vault.build_history`
The `skynet_vault` database stores high-fidelity build telemetry for the T2BM pipeline and autonomous builder coordination.
- **Purpose:** Enables spatial queries for the `NPUSpatialEngine` to prevent build overlaps and track hardware performance (Hailo-8L vs Edge TPU).
- **Structure:**
  - `id`: Unique record ID.
  - `name`: Generated identifier (e.g., `hailo_8l_tower_1234`).
  - `x`, `y`, `z`: Origin coordinates.
  - `width`, `depth`: AABB dimensions.
  - `schematic_name`: Source `.schem` file.
  - `ai_hardware`: The accelerator responsible for the build.
  - `timestamp`: Record creation time.
