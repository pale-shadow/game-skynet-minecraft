\chapter{Distributed Node Topology}
\label{chap:topology}

The bitsmasher.net infrastructure is established on a decentralized, heterogeneous compute grid. Node assignment splits heavy I/O operations, JVM world-state execution, and on-device neural inferencing across optimized platforms.

\section{Core Compute Grid Nodes}

\begin{description}
    \item[Chonk] The primary gameplay execution engine. It hosts the Paper server and coordinates low-level coordinate mutations via custom bridge interfaces.
    \item[Skynet] The local AI orchestration node. It processes neural pipeline coordination, NBT file manipulation, and runs the local spatial engines.
    \item[Stargate] The high-throughput Model Context Protocol (MCP) hub and network routing plane.
    \item[Edge-T] The physical network boundary agent handling telemetry interception.
\end{description}
