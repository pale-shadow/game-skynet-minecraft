☁️ Alphabet Service Integration Protocol
This protocol ensures that our local NPU/TPU acceleration tier can securely request creative reasoning from the cloud while staying within our $10/mo credit budget.

1. Host-Level Installation: gcloud CLI
The gcloud binary must be installed on Stargate (the bridge) and Skynet (the orchestrator). While edge nodes like Edge-T and Edge-J usually don't talk to GCP directly, having the CLI available aids in telemetry debugging.

Installation Steps (Debian/LMDE):

# Add the Google Cloud SDK distribution URI as a package source
echo "deb [signed-by=/usr/share/keyrings/cloud.google.gpg] https://packages.cloud.google.com/apt cloud-sdk main" | \
    sudo tee -a /etc/apt/sources.list.d/google-cloud-sdk.list

# Import the Google Cloud public key
curl https://packages.cloud.google.com/apt/doc/apt-key.gpg | \
    sudo gpg --dearmor -o /usr/share/keyrings/cloud.google.gpg

# Update and install
sudo apt update && sudo apt install google-cloud-cli



2. Authentication Flow
To allow our Python scripts (running in .venv) to access Vertex AI without hardcoding API keys, we use Application Default Credentials (ADC).

User Authentication: Used for manual management.

gcloud auth login

Service Authentication (ADC): This is what the VertexBridge logic uses.

gcloud auth application-default login

Note: Run this from within the Stargate terminal. It generates a JSON file in ~/.config/gcloud/ that the Google AI Platform SDK automatically detects.

3. Service Configuration: Vertex AI Reasoning
The Stargate Control Plane acts as the singular gateway. This minimizes the attack surface and allows for centralized credit monitoring.

Service: aiplatform.googleapis.com (Vertex AI)

Model: gemini-1.5-pro (for complex architectural reasoning)

Fallback Model: Local Ollama:Llama3 on Chonk (activated when credits drop below $1.00).

Implementation in src/internal/vertex_bridge.py
The code is designed to be Context-Aware. It doesn't just ask for "a building"; it passes the classification results from our Edge-T (Coral TPU) vision sensor to the cloud.
