This mod provides customizable login messages.

[https://www.curseforge.com/minecraft/mc-mods/miraisoftloginmessages](https://www.curseforge.com/minecraft/mc-mods/miraisoftloginmessages)

The messages are displayed to a user after server login.

 

Usage examples:

/loginmessage add Welcome to the server!
/loginmessage list
/loginmessage remove 1
/loginmessage removeall
/loginmessage help
/loginmessage add <<c>>Red text

 

If the mod is installed on server and client, there will be an autocompletion for the command arguments.
The mod supports easier text formatting now. For example, use <<c>> instead of §c to get red text (you cannot insert the § symbol in the chat window).
