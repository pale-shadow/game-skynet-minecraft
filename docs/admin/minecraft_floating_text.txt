## Floating Text 

Enable command blocks in server.properties file. Stop/Start server

place cmd block about 1 or 2 high, with a lever next to it

/summon armor_stand ~ ~ ~ {Invisible:1b,Invulnerable:1b,NoGravity:1b,Marker:1b,CustomName:"{\"text\":\"This way to Train Station\",\"color\":\"aqua\"}",CustomNameVisible:1b}

Command to kill the nearest text to you:
/kill @e[type=minecraft:armor_stand,sort=nearest,limit=1]
