# PROJECT: DREAMLAND STARGATE MCP
**Date Architecture Established:** March 2026
**Theme:** Cyber-Noir / William Gibson (Neuromancer)
**Core Objective:** Distributed Text-to-Building Minecraft (T2BM) via AI acceleration.

---

## 🏗 System Architecture (Multiprocessor Control Plane)
The project operates across a distributed mesh of hardware "Constructs" managed by the Stargate MCP.

### 1. Stargate MCP (Central Orchestrator)
- **Role:** The "Neuromancer" logic core. Dispatches high-level architectural intent to remote hosts.
- **Service:** `stargate-daemon.py`
- **Location:** `/home/minecraft/servers/stargate`

### 2. Skynet Host (Primary Fabricator)
- **Hardware:** Raspberry Pi 5 + Hailo-8L NPU + Google Edge TPU (USB).
- **Specialty:** High-density geometric expansion. Uses Hailo for "Muscle" (voxels) and USB TPU for "Sight" (style-check).
- **Zone:** Skynet AI Testing Field (Hubs 01-07).

### 3. Edge-T Host (Tactical Construction)
- **Hardware:** ASUS Tinker Edge T (SBC) with Integrated Edge TPU.
- **Specialty:** Reactive mutation and perimeter vision. Handles low-latency terrain verification.
- **Zone:** Neural Uplink & Solar Induction (Peripheral Sprawl).

---

## 🛠 T2BM Interlayer Protocol
The build system uses "Intent-based" construction rather than coordinate-mapping.
- **Foundation:** `base_plate` -> Deepslate Tiles.
- **Structure:** `skeleton` -> Polished Tuff.
- **Energy:** `reactor` -> Crying Obsidian / Copper Bulbs.
- **Aesthetic:** "Void-Tech" (Purple/Cyan/Tuff palette).

---

## 🛰 Infrastructure & Markers
- **BlueMap:** Visualizing the Sprawl in real-time.
- **Deep Rail Network:** Automated tunnel boring connecting Hubs.
- **ICE Level:** Server-side permissions (EssentialsX) protecting data vaults.

## 📜 Directives for AI Agents
- Maintain the "Sprawl" aesthetic: High-tech, low-life.
- Every construction must be validated by NPU/TPU hardware.
