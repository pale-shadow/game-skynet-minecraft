## 🎮 How to Join

### **Java Edition (Recommended)**
1.  Launch Minecraft **Java Edition 1.21.11**.
2.  Click on **Multiplayer** then **Add Server** [21, 22].
3.  Enter `minecraft.bitsmasher.net` in the Server Address field [17].
4.  Join and start your journey at **Washington Station (The Zero Point)** [23].

### **Bedrock Edition (Cross-Platform)**
1.  Open Minecraft and select the **Servers** tab [21, 24].
2.  Click **Add Server** and enter the following:
    *   **Server Name:** Bitsmasher
    *   **Server Address:** `minecraft.bitsmasher.net`
    *   **Port:** `19132` (Default)
3.  Click **Play** to enter the world
