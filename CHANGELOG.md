# Changelog

All notable changes to the Dreamland / Skynet project will be documented in this file.

## [2026.05.08] - Emerald Mirror Pivot

### Added
- **Emerald Mirror Pipeline**: Transitioned the distributed AI control plane to a greebling-focused spatial mutation engine.
- **v5 Industrial Standards**: Formalized high-fidelity architectural requirements, including fluted pillars and grid-iron girders.
- **NVIDIA Jetson Cluster (Edge-J)**: Integrated four Jetson nodes for CUDA-accelerated voxel processing.
- **Schematic Types**: Introduced `Foundation` and `Delta` classification for schematics to support layered AI builds.
- **Chronosplicer Singularity**: Deployed the first greebled generator delta based on the "Visionary" prose.

### Changed
- **Node Re-Tasking**:
    - **Skynet (Hub-00)**: Pivoted to Structural Integrity Audits and global orchestration.
    - **Stargate (Hub-01)**: Pivoted to Macro-Schematic Orchestration via Ollama.
    - **Edge-T (Hub-06)**: Pivoted to Greebling Density Verification.
- **Metadata Requirements**: Added mandatory `schematic_type` field to all schematic metadata JSONs.
- **RCON Throttling**: Standardized 0.05s per-block delay for all automated injections.

### Decommissioned
- **Void-Tech Logic**: Formally retired all mycelial-based architectural generation.
- **Legacy Mutation Scripts**:
    - `src/schematics/adaptive_mutation_v7.py`
    - `src/schematics/reactive_mutation.py`
    - `src/schematics/builders/void_tech.py`
    - `src/schematics/rail_yard_logic.py` (Vertical logistics replaced by flexible foundations).
- **Go Generation**: Retired legacy Go-based schematic generation tools in favor of Python/T2BM.
