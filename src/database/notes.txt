### 1. Configure Nginx as a Reverse Proxy

This is the most common setup. Nginx sits in front of BlueMap's internal web server (port 8100) and handles the traffic.

```nginx
server {

    # Increase timeout for large map data transfers

    proxy_connect_timeout 300;
    proxy_send_timeout 300;

    location / {
        proxy_pass http://10.10.8.60:8100; # The IP/Port of your BlueMap server

        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # Enable compression for faster loading

        gzip_types text/plain text/css application/json application/javascript;
    }
```
