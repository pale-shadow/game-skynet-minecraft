# 🏗️ Shroomville Urbanization: Phase 2.1 Manifest

This manifest contains the JSON prompts for the **Void-Tech** expansion. These structures are designed to be generated using the `mcschematic` Python engine and deployed via FAWE.

---

### 1. The Mycelial Uplink Spire (`void_uplink_v2.json`)
A sleek, needle-like tower that acts as the primary BlueMap data transmitter. It features a "floating" top section held by magnetic induction (End Rod pylons).

### 🛠️ Execution Note for The Janitor
When deploying these v2.1 schematics:
1.  **Block States**: Ensure your Python builder is passing the `variant=pearlescent` tag for the froglights.
2.  **Water-Logging**: If using the `growth_chamber_v2`, use `//paste -a` to preserve the water inside the cherry leaf blocks for the bio-bubbling effect.
3.  **BlueMap**: Update your `world.conf` to include these new POIs using the hex color `#a331d1` (Magenta) to differentiate Void-Tech from standard Industrial sites.
